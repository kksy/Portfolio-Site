HI THERE! I'M KYLIE,

a Web Developer with experience in building web applications using technologies such as HTML, CSS, JavaScript and Ruby on Rails.

I gained interest in basic web development, when I suggested to fix the layout of a student-run sports club website through HTML and CSS customisation. I expanded on this knowledge by making my crafts and food blog from scratch with the help of Learn Web Code on YouTube. Due to these reasons, along with my programming experience while doing my Electrical Systems major, I became even more fascinated with web development. In order to have structure while building up on my existing skills, I decided to enrol in General Assembly's Web Development Immersive programme.

PORTFOLIO

TIC TAC TOE

This is a Tic Tac Toe game, which was my first project in the course. It features a scoring system and customisable markers.

HTML5 CSS3 JavaScript jQuery UnderscoreJS

GITHUB LIVE DEMO

STRETCH

This app encourages users to try new things outside their comfort zone. The challenges (Stretches) are both system and user generated. If completed, users can choose to write a short entry on their experiences and upload an image.

HTML5 CSS3 JavaScript Bootstrap 3 Masonry Ruby Sinatra PostgreSQL

GITHUB LIVE DEMO

AGRA SHARE

Agra Share is a platform for users to rent out their agricultural equipment. Lenders can create a listing. Borrowers can reserve the item and contact the lender through email.

This was a team project. My role was to create a CRUD app for the listings, as well as front-end development.

HTML5 CSS3/Sass Ruby on Rails PostgreSQL jQuery/AJAX Foundation 5 Google Maps & SendGrid API

GITHUB LIVE DEMO

WDI CONFERENCE 2015

WDI Conference is a mock technology conference run by the students of General Assembly. We were assigned to create a website that displays information regarding this event.

This was a team project. My role was front-end development.

HTML5 CSS3/Sass jQuery

GITHUB LIVE DEMO

POKÉHUNT

In order to practice more NodeJS, I made PokéHunt, which is a treasure-hunting game created for mobile use. It features Google Maps API and the use of geolocation to simulate player adventure to catch Pokémon surrounding the map. This demo only has data for Melbourne CBD.

NodeJS ExpressJS PostgreSQL HTML5 CSS3 jQuery/AJAX Materialize Google Maps API PassportJS
